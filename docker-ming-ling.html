Docker 命令

Permalink: 2020-11-20 10:20:00+01:00 by hanf in docker tags: docker

镜像相关

docker images

查看docker镜像

docker build

通过Dockerfile文件创建镜像

docker save

将docker镜像打包成文件

docker load

加载文件，解析成镜像

容器相关

docker run

从镜像开启一个容器

docker exec

执行一个运行中的容器

docker start

开启一个停止状态的容器

docker stop

停止一个运行中的容器
