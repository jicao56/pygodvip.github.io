私钥/公钥/地址

  • 私钥：一个256位的随机数，发送以太坊交易时，用于创建签名 来证明自己对资金的所有权。
  • 公钥：一个512位(64字节)的数，由私钥通过椭圆曲线加密secp256k1算法单向生成的。
  • 地址：由公钥的Keccak-256算法单向hash，并取的最后20个字节（160位）派生出来的标识符。

安全须知

  • keystore文件就是加密存储的私钥。所以当系统提示你选择密码时：将其设置位强密码，备份并不要共享。如果你没有密码管理器，请将其写下来并锁在保险柜里。要访问账户，你必须同时有 keystore文件 和 密码。
  • 助记词可以导出私钥，所以可以人为助记词就是私钥。请使用笔和纸进行物理备份。不要吧这个任务留给”以后“，你会忘记。
  • 切勿以简单形式存储私钥，尤其是电子方式存储。
  • 不要将私钥用电子形式存储，如：电子文档/数码照片/屏幕截图等等。使用密码管理器 或者 物理备份。
  • 在转移大额金额之前，先做一个小额的测试交易。收到测试交易后，再尝试从该钱包发送。

以太币单位

  • 以太币的单位成为“以太”，也称为“ETH”，或符号“”
  • ether可以被细分为更小的单位，最小的单位为“wei”，1 ether = 10^18 wei
  • 以太的值总是在以太坊内部表示为 以wei表示的无符号整数
  • 以太的各个单位都有一个使用国际单位制（SI）的科学名称和一个口语名称。

以太坊钱包

进入以太坊系统的门户，包含了私钥，代表我们创建和广播交易。
- MetaMask：一个浏览器扩展钱包，可在浏览器中运行。
- Jaxx：一款多平台/多币种的钱包，可在各种操作系统上运行，包括Android/ios/windows/mac/linux。
- MEW：MyEtherWallet，一个基于web的钱包，可以在任何浏览器中运行。
- Emerald Wallet：旨在与ETC配合使用，但与其他基于以太坊的区块链兼容。

MetaMask

安装metamask

在chrome浏览器中，安装metamask扩展。

第一次使用metamask

会要求创建metamask密码，验证助记词等，按操作往下走即可

切换网络

  • Main Network（NetworkID：1）

    以太网的主网络。真正的ETH，真正的价值，真正的结果

  • Ropsten Test Network（NetworkID：3）

    以太坊公共测试区块链和网络，使用工作量证明POW共识（挖矿）。该网络上的ETH没有任何价值 。除了ETH没有价值外，其他一切都一样，可以认为是主链的副本。

  • Kovan Test Network（NetworkID：42）

    以太坊公共测试区块链和网络，使用 “Aura” 协议进行权威证明POA共识（联合签名）。该网络上的ETH没有任何价值。此测试网络仅由Parity支持。主要针对parity客户端。

  • Rinkeby Test Network（NetworkID：4）

    以太坊公共测试区块链和网络，使用 “Clique” 协议进行权威证明POA共识（联合签名）。该网络上的ETH没有任何价值。主要针对gas客户端。

  • Localhost 8545

    连接到与浏览器在同一台计算机上运行的节点。该节点可以是任何的公共区块链节点，也可以是任何的私有测试节点。

  • Custom RPC

    允许将metamask连接到任意兼容geth的RPC接口的节点。该节点可以是任何公共或者私人区块链的一部分。
